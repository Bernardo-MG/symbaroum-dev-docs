# Roles

The following roles are used to handle privileges:

|Role|Privileges|
|---|---|
|User|Can edit the data he created|
|Admin|Can edit any data|

## Privileges

Roles are composed of the following privilege rules:

* Edit own character
* Edit own ability
* Edit own trait
* Edit own monstrous trait
* Edit own boon
* Edit own burden
* Edit own power
* Edit own ritual
* Edit own race
* Edit own tradition

* Edit any character
* Edit any ability
* Edit any trait
* Edit any monstrous trait
* Edit any boon
* Edit any burden
* Edit any power
* Edit any ritual
* Edit any race
* Edit any tradition
